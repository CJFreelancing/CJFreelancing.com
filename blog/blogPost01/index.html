Welcome to Chris Jones Freelancing

Honestly I made this website strictly off the fact that I need a job post-graduation. What better way to show off what I learned from school and my skill sets then to put it all together to make a personal website?

If you couldn’t tell from the title, this website will also be used to find freelance work on the side outside of my main programming career. I don’t have any set prices for my work. If you need work completed it is best to contact me directly to discuss a price estimate.

Periodically I will post blogs about Computer Sciency and overall technology things. In the field of Software Development learning never stops. I plan to use blogging to keep up with current technology trends, complicated topics and to help other up-and-coming engineers.

Lastly but most importantly I will be posting material related to my portfolio on here. My portfolio will consist of posting code snippets and explaining in-depth about projects that I have completed. Most full projects will be posted on GitHub and if it’s not on there it is because I did work for a client therefore I do not want to post their source code. However if I did work for a client and they did not pay for my work, of course I will happily post their source code for all the web to have.
